# angularBlog

```
angular+node(express)+mongoDb
```

## #angularBlog博客的设计理念:主要运用了前后端的分离思想，实现了个人注册、个人登录功能、文章评论、用户个人资料以及账号信息的修改， 用户可以很好的管理个人的博客。发布博客，修改博客以及删除博客，实现了多用户博客。
前端部分：

下载依赖

```
cnpm install
```

运行：

```
npm start
```
打包：

```
npm build
```
后台部分：（agBlogServer）

下载依赖：

```
cnpm install
```

运行：

```
npm start
```
